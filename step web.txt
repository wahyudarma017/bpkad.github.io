# PORTAL LAYANAN ABSENSI BPKAD ENDE


-> Absen Karyawan 

Absen Karyawan - ABSENSI KARYAWAN merupakan aplikasi absensi berbasis web, yang dibuat untuk memenuhi kebutuhan instansi yang membutuhkan aplikasi absensi karyawan.

-> Fitur - fitur aplikasi  

1.admin dapat mengontrol aplikasi, mengelola data user, menambahkan, melihat, mengubah atau menghapus data.

2. ABSENSI MASUK DAN ABSENSI PULANG

3.ABSENSI KARYAWAN, sistem ini memungkinkan setiap karyawan melakukan absensi berdasarkan sesi nya masing - masing.  Untuk mencegah penyalahgunaan data.

-> Login default admin : 
  1. username = admin
  2. password = admin
  
-> Login default karyawan : 
  1. username = user
  2. password = user
  
-> Catatan(PENTING) :
   Sebelum mengimport database, pastikan anda membuat database sesuai dengan nama database yang ada di dalam folder db, yaitu 'karyawansi'
